# MTG Combinator

A site built using FastAPI and React that returns all the combos in your Magic: The Gathering Commander/EDH deck.

You can find the site at [mtgcombinator.com](https://mtgcombinator.com/)

## Depends on

MTG Combinator would not be possible without these great resources

### User Deck Data
- [Archidekt](https://archidekt.com)
- [Moxfield](https://moxfield.com)
- [MTGGoldfish](https://www.mtggoldfish.com)

### Card Data
- [Scryfall](https://scryfall.com)
### Combo Data
- [Commander Spellbook](https://commanderspellbook.com/)
